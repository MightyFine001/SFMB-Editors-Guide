What is Mario Multiverse?

Mario Multiverse Super Fanmade Bros.: is a program developed by neoarc. Allowing users to create Super Mario levels in a simmilar vein as Super Mario Maker 2. Mario Multiverse is currently in closed beta, but a public demo is available as of now. The community surronding this game continues to grow everyday, contributing to this project, and so can you!

Download the Mario Multiverse demo through our community discord server today! (Only available for windows computers)
